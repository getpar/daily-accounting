# 每日小记 📒

个人记账应用 — 记录每一笔收支，掌握每一分去向。

支持 **Windows 桌面** 和 **微信小程序** 双端使用，数据本地存储，安全可靠。

---

## ✨ 功能特性

### 🖥️ Windows 桌面版

| 模块 | 功能 |
|------|------|
| 📝 记账 | 支出/收入双类型，金额+日期+二级分类+备注 |
| 📋 账单 | 按月份/类型筛选，关键词搜索，编辑删除，批量删除 |
| 📊 统计 | 饼图、柱状图、折线图，月度收支对比 |
| 💰 预算 | 月度预算设置，进度条颜色预警，超支横幅提醒 |
| 📥 导出 | Excel 导出（支持按月份和类型筛选） |
| 🏷️ 分类 | 10个一级大类+50+二级小类，支持自定义添加删除 |
| 🔁 周期账单 | 房租/订阅等固定账单，手动触发+自动更新下次日期 |
| 🌙 暗色模式 | 侧边栏一键切换亮色/暗色 |
| ⌨️ 快捷键 | 默认 Ctrl+Shift+N 快速记账，支持自定义 |
| 📦 托盘 | 最小化到系统托盘，右键菜单，单实例运行 |
| 💾 备份恢复 | 一键备份数据库，从备份恢复 |

### 📱 微信小程序版

| 模块 | 功能 |
|------|------|
| 📝 记账 | 支出/收入，金额+日期+二级分类+备注 |
| 📋 账单 | 月份/类型筛选，关键词搜索，编辑删除 |
| 📊 统计 | 分类占比条+每日趋势，月份切换 |
| 💰 预算 | 月度预算设置，进度条颜色预警 |
| 📥 导出 | Excel 导出（.xlsx 文件预览） |
| 🏷️ 分类 | 10个一级大类+50+二级小类，自定义添加删除 |
| 📅 月份 | 首页月份选择器，查看历史数据 |

## 🚀 快速启动

### 桌面版

```bash
cd desktop
npm install
npm run dev
```

> 数据存储位置：`%APPDATA%/daily-notes/daily-notes.db`

### 微信小程序版

```bash
cd miniapp
npm install
npm run build:weapp
```

> 用微信开发者工具打开 `miniapp/dist/` 目录即可预览

## 🛠️ 技术栈

| 端 | 技术 |
|------|------|
| Windows 桌面 | Electron + React + TypeScript + Ant Design |
| 微信小程序 | Taro 3.x + React + TypeScript |
