# 每日小记 📒

个人记账应用 — 记录每一笔收支，掌握每一分去向。

支持 **Windows 桌面** 和 **微信小程序** 双端使用，数据本地存储，安全可靠。

---

## ✨ 功能特性

### 🖥️ Windows 桌面版

| 模块 | 功能 |
|------|------|
| 📝 记账 | 支出/收入双类型，金额+日期+二级分类+备注 |
| 📋 账单 | 按月份/类型筛选，关键词搜索，编辑删除，批量删除 |
| 📊 统计 | 饼图、柱状图、折线图，月度收支对比 |
| 💰 预算 | 月度预算设置，进度条颜色预警，超支横幅提醒 |
| 📥 导出 | Excel 导出（支持按月份和类型筛选） |
| 🏷️ 分类 | 10个一级大类+46个二级小类，支持自定义添加删除 |
| 🔁 周期账单 | 房租/订阅等固定账单，手动触发+自动更新下次日期 |
| 🌙 暗色模式 | 侧边栏一键切换亮色/暗色 |
| ⌨️ 快捷键 | 默认 Ctrl+Shift+N 快速记账，支持自定义 |
| 📦 托盘 | 最小化到系统托盘，右键菜单，单实例运行 |
| 💾 备份恢复 | 一键备份数据库，从备份恢复 |

### 📱 微信小程序版

| 模块 | 功能 |
|------|------|

## 🚀 快速启动

### 桌面版

```bash
cd desktop
npm install
npm run dev
```

> 数据存储位置：`%APPDATA%/daily-notes/daily-notes.db`
>
> 打包为可安装程序：`npm run build`（产物在 `desktop/release/` 目录）
>
> ⚠️ 如启动报错 `Cannot read properties of undefined (reading 'whenReady')`，说明系统环境变量中 `ELECTRON_RUN_AS_NODE` 被设为了 `1`，在系统环境变量中删除该项后重新启动即可。

### 微信小程序版

> ⚠️ **前置要求**：Taro 3.x 需要 **Node.js v20**，系统默认 v24 不兼容。需先切换 Node 版本：
> ```bash
> set PATH=%USERPROFILE%\node-v20\node-v20.18.0-win-x64;%PATH%
> ```

```bash
cd miniapp
npm install
npm run build:weapp
```

> 用微信开发者工具打开 `miniapp/dist/` 目录即可预览
>
> ⚠️ **编译前必须关闭微信开发者工具**，否则 `dist/` 目录被锁，新代码写不进去。
